Exemplo de aplicação com Broadcast Channel e Web Crypto API

Neste exemplo os dois microfrontends estão totalmente desacoplados. As duas aplicações, apesar de isoladas, podem publicar e escutar mensagens no BroadcastChannel.

Microfrontend 1 - Enigma

Este microfrontend é uma aplicação Angular, que faz alusão ao equipamento criptográfico muito utilizado na II Guerra Mundial, o Enigma. Nela utiliza-se a Web Crypto API para gerar uma chave simétrica e criptografar uma mensagem, posteriormente publicada no BroadcastChannel e a disposição de todos outros componentes presentes no mesmo contexto

Microfrontend 2 - Turing

Este microfrontend é uma aplicação Angular, que faz alusão ao matemático Alan Turing, que foi um dos responsáveis por quebrar o código da máquina Enigma. Nela utiliza-se a Web Crypto API para descriptografar a mensagem publicada no BroadcastChannel e exibir o resultado na tela
